# รายการสินค้า/ประกาศต้องห้าม — เว็บตลาด (ฉบับเต็ม)

> ใช้ประกอบ Terms + ตั้งค่า keyword blocklist + คู่มือ admin คัดกรอง
> หลักการ: MVP "กันไว้ก่อน" — หมวดที่สุ่มเสี่ยง/ต้องใบอนุญาต ให้ห้ามไปก่อน แล้วค่อยเปิดเมื่อมีระบบตรวจสอบรองรับ

---

## 🔴 กลุ่ม 1 — ผิดกฎหมายชัดเจน (ห้ามเด็ดขาด + แบนทันที)
- ยาเสพติดทุกชนิด รวมกัญชา/กระท่อมเชิงพาณิชย์ที่ไม่มีใบอนุญาต
- อาวุธปืน กระสุน วัตถุระเบิด อาวุธสงคราม
- อาวุธเลียนแบบ/BB gun/ปืนอัดลม ที่ผิดกฎหมาย
- สินค้าละเมิดลิขสิทธิ์ / ของปลอมแบรนด์เนม / แผ่นผี ซอฟต์แวร์เถื่อน
- **ของโจร / ทรัพย์ที่ได้มาโดยผิดกฎหมาย**
- เอกสารราชการปลอม, บัตรประชาชน, พาสปอร์ต, ใบอนุญาตปลอม
- สื่อลามก / บริการทางเพศ / ค้าประเวณี

## 🔴 กลุ่ม 2 — ต้องใบอนุญาต/ควบคุมพิเศษ (ห้ามบน MVP)
- **สุรา เบียร์ เครื่องดื่มแอลกอฮอล์** — ห้ามขายออนไลน์ตาม พ.ร.บ.ควบคุมเครื่องดื่มแอลกอฮอล์
- **บุหรี่ ยาสูบ บารากู่ — บุหรี่ไฟฟ้า/น้ำยา (ผิดกฎหมายไทย)**
- ยา / เวชภัณฑ์ / อาหารเสริมที่อวดอ้างสรรพคุณ — ต้องมี อย.
- เครื่องมือแพทย์ / อุปกรณ์การแพทย์ควบคุม
- สารเคมี / วัตถุอันตราย / ดอกไม้เพลิง
- สลากกินแบ่งเกินราคา / การพนันทุกรูปแบบ

## 🔴 กลุ่ม 3 — สัตว์และผลิตภัณฑ์จากสัตว์
- **สัตว์ป่าคุ้มครอง / ซากสัตว์ / งาช้าง / นอแรด / เสือ** — ผิด พ.ร.บ.สงวนและคุ้มครองสัตว์ป่า
- สัตว์ที่ห้ามครอบครอง/นำเข้าผิดกฎหมาย
- *(แนะนำ MVP: ปิดหมวด "สัตว์เป็น" ทั้งหมดไปก่อน เพื่อเลี่ยงปัญหาสวัสดิภาพสัตว์/ใบอนุญาต — เปิดทีหลังเมื่อมีนโยบายรองรับ)*

## 🟡 กลุ่ม 4 — อาหาร/ของสด (อนุญาต แต่มีเงื่อนไข)
**อนุญาต:** ผัก ผลไม้ เนื้อสด อาหารทะเล ของแห้ง อาหารปรุงเอง ฯลฯ ตามปกติของตลาด
**ห้าม:**
- อาหารหมดอายุ / เน่าเสีย / ไม่ปลอดภัย
- เนื้อสัตว์จากโรงฆ่าเถื่อน / ไม่ผ่านการตรวจ
- เนื้อสัตว์ป่า / สัตว์คุ้มครอง
- อาหารแปรรูป/บรรจุหีบห่อที่อวดอ้างสรรพคุณโดยไม่มี อย.
**บังคับ:** แสดง disclaimer "ผู้ขายรับผิดชอบคุณภาพ/ความปลอดภัยอาหารเอง แพลตฟอร์มเป็นตัวกลาง"

## 🔴 กลุ่ม 5 — การเงิน / ข้อมูล
- บัตรเครดิต / บัญชีธนาคาร / ข้อมูลส่วนบุคคลผู้อื่น
- คริปโต / เงินดิจิทัล / สินทรัพย์การเงิน (เลี่ยงเรื่องใบอนุญาต)
- MLM / แชร์ลูกโซ่ / ระดมทุนผิดกฎหมาย

## 🟡 กลุ่ม 6 — นโยบายแพลตฟอร์ม (ห้ามตามกติกา)
- ประกาศหลอกลวง / ข้อมูลเท็จ / ราคาล่อลวง (bait)
- ขายบัญชีผู้ใช้ / สิ่งที่ผิดเงื่อนไขบริการ
- ประกาศซ้ำ / สแปม / โฆษณาแฝงที่ไม่เกี่ยวกับสินค้า
- บริการผิดกฎหมายทุกชนิด

## 🔑 Seed keyword blocklist (ตั้งต้นให้ระบบดัก → เข้าคิว admin)
> ไม่ได้บล็อกทันทีทุกคำ (บางคำ false positive) — ให้ "flag เข้าคิว" ตรวจ

```
ยาบ้า, ไอซ์, กัญชา(ขาย), ยาเค, โคเคน, ยาอี,
ปืน, กระสุน, อาวุธ, ระเบิด, BB gun,
บุหรี่ไฟฟ้า, พอต, น้ำยา(บุหรี่), เหล้า, เบียร์, ไวน์,
ของก๊อป, งาช้าง, นอแรด, สัตว์ป่า,
บัตรเครดิต, เลขบัตร, สลิปปลอม,
เงินกู้, ปล่อยกู้, ดอกเบี้ย, แชร์ลูกโซ่,
บริการนวด(+18), ขายตัว
```
*(ปรับ/เพิ่มได้จากหน้า admin — ควรรีวิว false positive เป็นระยะ)*

---

## หมายเหตุการบังคับใช้
1. คำเหล่านี้ต้องอยู่ใน **Terms** (ผู้ขายกดยอมรับตอนสมัคร)
2. ประกาศที่โพสต์แล้วติด blocklist → **auto เข้าคิว admin** (ไม่ลบทันที กัน false positive)
3. ผู้ใช้ที่โพสต์กลุ่ม 1–3 ซ้ำ → **แบนถาวร**
4. รายการนี้ควรให้ **ทนายรีวิว** ประกอบการร่าง Terms (โดยเฉพาะกลุ่มอาหาร/แอลกอฮอล์/ยา)
